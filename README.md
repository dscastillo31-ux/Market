# Market

Aplicación web progresiva (PWA) para el control completo de una tienda de abarrotes, frutas y verduras.

## Funcionalidades
- Venta rápida con escáner de código de barras
- Control de inventario por unidad y por peso (frutas/verduras en gramos)
- Control de fiado con abonos parciales
- Cierre de caja diario
- Análisis de rotación de productos
- Control de merma y pérdidas
- Gestión de proveedores y cuentas por pagar
- Gastos fijos y variables con presupuesto
- Comparación entre períodos
- Exportación CSV para Power BI
- Funciona sin internet (PWA instalable)

## Tecnologías
HTML · CSS · JavaScript · Chart.js · ZXing (escáner)
